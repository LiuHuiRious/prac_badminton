# prac_badminton


## 使用jasmine进行单元测试



### 测试1：

当给一串不符合规定的字符串时返回 "Error: the booking is invalid"



### 测试2：

当给一串不是整点的时间时，返回 "Error: the booking is invalid"



### 测试3：

当给一串符合规定且时间时整点的字符串时，返回 'Success: the booking is accepted'



### 测试4：

当给一个与已经预定的订单相同的字符串时，返回 'Error: the booking conflicts with existing bookings'



### 测试5：

当给一个错误的取消的订单的字符串时，返回 'Error: the cancel order was not right'



### 测试6：

当给一个正确的取消的订单的字符串时，返回 'Success: the booking is accepted'



### 测试7：

当给一个与已经预定的时间场地冲突的字符串时，返回 'Error: the booking conflicts with existing bookings'
